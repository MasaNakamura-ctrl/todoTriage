## 概要
![フロントエンド](./todoTriage.png)
Dockerのコンテナ上にUbuntuを構築し、その上にViteでReactの環境を構築しました。

それをベースに、ToDoアプリの拡張版としてトリアージアプリの実装をしました。

具体的にはタスクを4つの優先度(高いものから赤、黄、緑、黒)に分け、タスク追加したところから順を追って優先度を下げられるようにしています。
